✏️ 시험 문제

아래 시험문제를 다운받아 구현해야 하는 기능들을 구현하세요.
총 2개의 문제가 있습니다.

1. VS Code에 다운받은 파일을 불러와 엽니다.
2. npm 또는 yarn install로 프로젝트 종속성을 설치합니다.
3. npm 또는 yarn start 명령어로 프로젝트를 실행할 수 있습니다.

```
eg.)
yarn install
yarn start
```

💡 Redux toolkit의 “thunk”와 비동기 함수를 이용해서 프로젝트를 완성해봅니다.
